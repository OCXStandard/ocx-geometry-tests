# ocx-geometry-tests
> ### A python CLI project

``ocx-geometry``  provides Python scripts for creating 3Docx models with different geometry attributes for interoperability testing.

## Installation

install from pypi:
````commandline
pip install ocx_geometry_tests
````

# Usage
## The main CLI

````commandline
 ocx-geometry --help

````
